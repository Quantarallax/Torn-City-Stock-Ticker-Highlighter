# Torn-City-Stock-Ticker-Highlighter
Highlights a single Stock listing of your choice.

Install into TamperMonkey https://www.tampermonkey.net/

Enter your stock of choice into the box at the top e.g. MCS

<img width="313" height="132" alt="image" src="https://github.com/user-attachments/assets/66d65995-43a8-47e4-9edd-ad82b21c2ad1" />

MCS is then highlighted in the list.

<img width="1208" height="368" alt="image" src="https://github.com/user-attachments/assets/e0afb0be-a87a-4718-911b-ba0ae45fd2fc" />

I put this together as I tend to have one stock I'm building towards, but want to see others in the list. The ordering can sometimes be all over the place, so this allows me to zero in on the stock I'm currently building.
