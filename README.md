# Torn-City-Stock-Ticker-Highlighter
Highlights a single Stock listing of your choice.

Install into TamperMonkey https://www.tampermonkey.net/

Enter your stock of choice into the box at the top e.g. MCS

MCS is then highlighted in the list.

Reason I made this is that I tend to have one stock I'm building towards, but want to see others in the list. The ordering can sometimes be all over the place, so this allows me to zero in on the stock I'm currently building.
